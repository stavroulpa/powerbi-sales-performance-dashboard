#  Power BI Sales Performance Dashboard

## Overview
This project is an **Power BI case study** focused on analyzing sales performance and profitability across customers, products and countries.

The report supports **two different user groups**:
- Executive leadership (CEO & managers)
- Marketing and product specialists


## Key Insights
- Profit and loss drivers across countries, products, and categories
- Revenue and profit trends over time (YTD & YoY)
- Profit margin analysis by category, subcategory, and product
- Customer performance insights, including average profit per customer and top customers
- Drillthrough and tooltip-enabled exploration for deeper analysis


## Dashboards
- Executive Summary (CEO & Management)
- Sales & Profit Overview
- Customer Analysis
- Product & Category Insights
- Order Details (Drillthrough)


## Data Model
- Star schema design
- Fact table: Sales 2017-2020
- Dimension tables: Date, Customer, Product, Sales Territory
- Central Date table with full time intelligence support


## Tools & Skills
- Power BI Desktop
- Power Query
- DAX (YTD, YoY, Profit Margin)
- Data Modeling & BI Storytelling


## Notes
This project demonstrates the ability to translate business requirements into clean, executive-ready BI dashboards.
